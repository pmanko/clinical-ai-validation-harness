# Validation Harness Constitution

The canonical project constitution lives at
`.specify/memory/constitution.md`.

This durable docs entry remains as a pointer for readers browsing `docs/`.
The constitution defines the non-negotiable rules for:

- real production validation paths,
- deterministic reviewed transforms,
- record-level evidence and decision rationale,
- metadata, provenance, and traceability,
- behavior-defining tests with diverse scenarios,
- clinical evidence and operating metadata boundaries,
- PCCP-style change records for material validation changes.
